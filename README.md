# Legacy Performing Arts Website Starter

Static GitHub Pages-ready site using the Legacy Performing Arts branding.

## Pages
- `index.html` — Home, classes preview, and contact CTA
- `about.html` — About the studio
- `payments.html` — Payment buttons and Stripe placeholder links

## What changed in this version
- Header logo now uses a transparent PNG so no solid background box shows over the transparent/dark hero header.
- Homepage has a dark, performance-inspired hero section with the Legacy color palette.
- Navigation is mobile responsive.

## Setup
Upload all files to a GitHub repository and enable GitHub Pages.

## Stripe Notes
Replace the placeholder Stripe links on `payments.html` with real Stripe Payment Links.

For a Stripe Customer Portal, add a backend later with Cloudflare Workers/Pages Functions, Netlify Functions, or Vercel Functions. Do not put Stripe secret keys in this static site.
